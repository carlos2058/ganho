# Mini FF Arena

Protótipo de jogo 2D inspirado em battle royale (estilo FF), feito em HTML5 Canvas e JavaScript puro.

## Como rodar

```bash
python3 -m http.server 8000
```

Depois abra `http://localhost:8000` no navegador.

## Controles

- **WASD**: movimentação
- **Mouse**: mira
- **Clique esquerdo**: tiro
- **R**: recarregar
- **Espaço**: iniciar/reiniciar partida

## Objetivo

Elimine todos os inimigos enquanto permanece dentro da zona segura, que encolhe com o tempo.
